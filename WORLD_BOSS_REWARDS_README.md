# ワールドボス報酬制度改定 2026

## 概要
ワールドボスの報酬システムを全面的に改定し、ボスレベルに応じて段階的に報酬の種類と量が増加するように設定しました。

## 変更内容

### 報酬の段階的スケーリング
- **Lv10-20**: 基本資源（食料、木材、石材）と準基本資源（青銅、薬草など）
- **Lv30-40**: 準基本資源の種類を拡大（布、馬、包帯、鉄、ガラス、大理石、硫黄、金、医薬品）
- **Lv50-60**: 貴重資源の追加（宝石、鋼鉄、石炭、知識、香辛料）+ 火薬資源開始（火薬、石油）
- **Lv70-80**: 火薬資源の拡大（文明クリスタル、ゴム、マナ、電子部品、チタン）+ 超貴重資源開始（ウラン）
- **Lv90**: 超貴重資源の拡大（ダイヤモンド、プルトニウム、シリコン、レアアース）
- **Lv100**: 弩級貴重資源追加（量子結晶、AIコア、遺伝子サンプル、ダークマター、反物質）

### 中間レベルボス
以下のボスに報酬を追加しました：
- Lv5: ゴブリン・ダガス
- Lv15: 悪魔・モルゲート
- Lv25: アイガオン
- Lv35: アンデバブ
- Lv45: バクスト
- Lv55: オセアノの蛇
- Lv65: トロールの化身
- Lv75: コンデット
- Lv85: ビースト軍団
- Lv95: エピックゴーレム

### ベテランボス
Lv10-100の全てのボスにベテラン版の報酬を設定しました。
- **コイン**: 通常ボスの2倍
- **資源**: 通常ボスの2倍
- **クリスタル・ダイヤモンド**: 通常ボスと同じ

## 順位別報酬

各ボスで以下の順位区分で報酬が配布されます：
- **1位**: 最大報酬
- **2-3位**: 1位の約50%
- **4-10位**: 1位の約20-40%
- **11-50位**: 1位の約5-10%

## 適用方法

```bash
mysql -u [username] -p [database_name] < world_boss_rewards_overhaul_2026.sql
```

## 注意事項

- このスクリプトは既存の `world_boss_rewards` テーブルの全データを削除してから新しい報酬を設定します
- 実行前に必ずバックアップを取ってください
- CRONジョブは不要です（報酬はボス討伐時/タイムアウト時に自動配布されます）
- user_id は INT 型です

## 環境への配慮

基本資源は多めに配布しますが、希少資源は環境を破壊しない程度に抑えています。
これにより、ゲーム内経済のバランスを保ちつつ、プレイヤーに適切な進行感を提供します。
